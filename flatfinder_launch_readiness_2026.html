⚡ Activation — What to do right now
PLAN A — SUPABASE
1
Go to supabase.com → New project → Region: ca-central-1
2
Create table beta_signups with columns: id, email, plan, city, bedrooms, budget, created_at
3
Copy your anon public key from Settings → API
4
Paste key into beta-signup.html. Replace YOUR_SUPABASE_URL
5
Set up Resend.com → verify flat@flatfinder.io → get API key → enable in Supabase Auth
PLAN B — FORMSPREE (START HERE)
1
Go to formspree.io → Sign up free (no credit card)
2
Click + New Form → Name it "FlatFinder Beta Signup"
3
Copy your endpoint URL — e.g. https://formspree.io/f/abcdefgh
4
Open beta-signup.html → Find YOUR_FORMSPREE_ID → Replace with your ID
5
Done. Test it. You'll get an email at flat@flatfinder.io for every signup.
PLAN C — EMAILJS
1
Go to emailjs.com → Sign up → Connect Gmail service
2
Create email template: Subject "New FlatFinder Signup" → Map fields (email, plan, city)
3
Copy your Public Key + Service ID + Template ID
4
Add EmailJS SDK to beta-signup.html + call emailjs.send() on submit
5
Every signup → email to flat@flatfinder.io. No database. You manage the list manually.
